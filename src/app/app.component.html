Welcome to {{ title }}!

Follow below steps for integration:

  • Create an angular app using angular cli:
    $ng new angular-circleci-firebase

  • Create a New Repository:
    Login into your GitHub account and go to your repositories and create new.

  • Enable git integration in our project by following below steps:
    • $cd your-project-folder
    • $git init
    • $git add .
    • $git commit -m "first commit"
    • $git remote add origin "repository URL generated above after creating repository."
    • $git push -u origin master

  • Deploy to Firebase:
    • First you need to create a project in Google Firebase, go to Google Firebase console and setup the project.

    • Install Firebase Tools


      Install Firebase tools globally using below command:

      $ sudo npm install -g firebase-tools

    • Configure Firebase

      Now login Firebase with your Google account using below command. This command will redirect to Google account login page.

      $ firebase login

    • Initialize Firebase

      Go to your angular2 project folder and execute following command.

      $ firebase init

      For below question give directory name as dist.
      What do you want to use as your public directory? dist

    • Production Build


      in angular.json Go to your project and modify the outputPath value to dist.
      "outputPath": "dist/your-project-name"
      to
      "outputPath": "dist",
      Use below command to build the final build, this will generate dist folder with all required packages. $ ng build --prod

    • Firebase Push

      $ firebase deploy

      Wait for deploy complete message.
      Will get the below URLs in the end:
      Project Console: https://console.firebase.google.com/project/angular-circleci-firebase/overview Hosting URL: https://angular-circleci-firebase.firebaseapp.com
